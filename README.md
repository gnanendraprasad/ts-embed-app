# Angular ThoughtSpot Embed App

This project demonstrates embedding ThoughtSpot visualizations in an Angular application.

Create an `.env` file in the project root and provide the ThoughtSpot host details for the embed SDK:

```env
TS_HOST=https://your-thoughtspot-host
```

## Scripts

- `npm start` – Run the development server using `ng serve`.
- `npm run build` – Build the application for production.
- `npm test` – Run tests with `ng test`.

Note that package installation requires internet access which may not be available in this environment.
